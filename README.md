###  Tic Tac Toe in Nim

Just a Tic Tac Toe game created in Nim

[Demo (JS version)](https://doccaico.github.io/tic-tac-toe-nim/)

### Build
```sh
# JS
$ nim js -d:Release -o:game.js main_js.nim
$ chrome docs/index.html

# terminal
$ nim c -d:Release main_terminal.nim
$ ./main_terminal
```
